Preguntas frecuentes

¿Hay descuentos disponibles? +

Sí, ofrecemos descuentos para familias numerosas y hermanos. Consulta nuestra sección de descuentos para más detalles.

¿Qué incluye el precio? +

El precio incluye todas las actividades, comidas y materiales necesarios para el campamento.

¿Puedo pagar a plazos? +

Sí, ofrecemos opciones de pago a plazos. Contáctanos para más información.

¿Qué sucede si mi hijo no puede asistir algunos días? +

Si tu hijo no puede asistir, contáctanos para reprogramar o ajustar el pago.

¿Cuándo se cierra el plazo de inscripción? +

El plazo de inscripción cierra el 15 de septiembre. ¡No te quedes fuera!

¿Puedo cambiar el menú de mi hijo una vez inscrito? +

Sí, puedes cambiar el menú contactando a nuestro equipo antes del inicio del campamento.
